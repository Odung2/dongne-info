# 동네정보 (Dongne Info)

> 우리 동네에서 벌어지는 일을 쉽게 알려주고, 주민이 목소리를 낼 수 있게 해주는 서비스

---

## 한 줄 정의

서울시 강남구·마포구의 재개발·재건축·도시계획 공고를 자동 수집하고,
AI가 쉽게 해석해서 주민에게 알려주고, 주민 의견을 모아주는 플랫폼.

---

## 왜 만드냐

### 문제
- 집값에 영향을 주는 행정 정보(재개발, 용도변경, 도시계획 등)는 공개되어 있음
- 근데 아무도 안 읽음 → 행정 언어라 어렵고, 어디서 찾는지도 모름
- 서울도시공간포털이 알림 서비스를 만들었지만 아무도 안 씀 → UX 최악, 해석 없음
- **읽는다 해도, 할 수 있는 게 없음** → 읽고 끝, 참여 불가

### 해결
- 공공 API로 데이터 자동 수집
- Claude API로 행정 용어를 쉬운 말로 요약 (예측이 아닌 팩트 요약)
- 카카오 알림톡으로 주민에게 전달
- **주민이 관심 표시하고 의견을 남길 수 있는 참여 구조**

### 왜 지금이냐
- 6월 3일 지방선거 → 후보 의정활동 연결 가능
- 선거 시즌 트래픽 피크 활용
- 선거 후 당선자 임기 추적으로 장기 서비스 전환

---

## 타겟 유저

**1차 타겟**: 강남구·마포구 거주 30~50대
- 집값에 민감
- 재개발 관심 있지만 정보 찾기 귀찮음
- 카카오톡 잘 씀

**2차 타겟**: 지역 언론사, 시민단체, 선거 캠프

---

## 핵심 가치

```
기존: "강남구 도시계획 결정고시 제2026-51호..."

우리: "강남구 삼성동 홍실아파트 재건축 정비구역으로 지정됐어요.
       → 뜻: 재건축 사업을 공식적으로 시작할 수 있게 됐다는 의미
       → 진행 단계: 정비구역 지정 (전체 7단계 중 2단계)
       → 관련 과거 사례: 은마아파트 (2024년 같은 단계 통과)

       👍 관심 847명  💬 의견 56개
       [이 건에 대해 의견 남기기 →]"
```

**원칙: 예측하지 않는다. 팩트를 쉽게 설명한다.**
- AI가 "집값 오른다/내린다" 예측 → 틀리면 신뢰 잃음
- 대신: 행정 용어 해석 + 진행 단계 안내 + 관련 사례 연결
- 유저가 스스로 판단할 수 있는 정보를 제공

## 뉴스레터(뉴닉)와 뭐가 다르냐

| | 뉴스레터 (뉴닉) | 동네정보 |
|---|---|---|
| 본질 | 미디어 | **플랫폼** |
| 콘텐츠 | 에디터가 뉴스 골라 씀 | **공공 데이터 자동 수집** |
| 관련성 | 세상 뉴스 (알면 좋은) | **내 아파트 공고 (몰르면 손해)** |
| 유저 역할 | 독자 (수동) | **참여자 (능동)** |
| 리텐션 | 콘텐츠 퀄리티 의존 | **내 동네 일이니까** |
| 바이럴 | 약함 | **"우리 아파트 건이야" → 공유** |
| 네트워크 효과 | 없음 | **같은 동네 사람 많을수록 가치 상승** |
| 원가 | 에디터 인건비 (연 14~15억) | **AI 자동화 (월 3~5만원)** |

### 네트워크 효과가 핵심

```
은마아파트 구독자 1명:   "관심 1명" → 의미 없음
은마아파트 구독자 100명: "관심 87명, 반대 62명" → 여론이 보임
은마아파트 구독자 500명: "주민 과반이 반대" → 구의회도 무시 못 함
```

뉴스레터는 구독자 1명이든 10만 명이든 콘텐츠 가치가 같지만,
동네정보는 **같은 동네 사람이 많이 쓸수록 서비스 가치가 올라간다.**

## 청원동의와 뭐가 다르냐

```
청원동의: "할 말 있으면 여기 와서 써라"    (빈 종이)
동네정보: "이거 나왔는데, 너는 어때?"       (대화)
```

| | 청원동의 | 동네정보 |
|---|---|---|
| 시작점 | 시민이 이슈 생성 | **데이터가 이슈를 가져옴** |
| 유저 부담 | 높음 (글 작성) | **낮음 (버튼 하나)** |
| 범위 | 국가 단위 | **내 동네** |
| 맥락 | 없음 (청원문만) | **AI가 설명해줌** |
| 연결 | 국회 (멀다) | **구의회 (가깝다)** |
| 체감 | "나라가 바뀌겠어?" | **"우리 아파트 건이잖아"** |

청원동의는 이미 각성한 시민을 위한 도구.
동네정보는 **보통 사람을 시민으로 만드는 도구.**
